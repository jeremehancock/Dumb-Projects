The Story Behind the Madness

Hey there! Welcome to my corner of the internet where "dumb" ideas somehow turn into pretty cool things. I started this collection because I kept building weird little web apps powered by AI, and my friends kept saying "you should put these somewhere!"

So here we are! These projects range from genuinely useful to "why did you make this?" but they're all built with love, curiosity, and probably too much coffee. Each one taught me something new about AI, web development, or just how to have fun while coding.
